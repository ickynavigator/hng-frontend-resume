### INSTRUCTIONS

##### Clone repo

#### CLIENT

1. Edit `/client/src/constants.js` file with the details you want on the site

1. Create an `/client/src/.env` file and add your faunadb secret for your database used in the contact us page `FAUNADB_SECRET = `

1. `cd client`

1. `yarn install`

1. `yarn run start`
